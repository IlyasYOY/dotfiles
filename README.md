# dotfiles

Это простой репозиторий, где я храню свои *dot* файлы.

## Настройка

Чтобы установить не все (что мне было не лень описывать) настройки, надо сделать:

```bash
python3 -m scripts
```

Для работы с *telescope* требуется *ripgrep*.
